力量 · 悬停 · 平衡

我的灵感来自于一个关于「力量与平衡」的身体瞬间。当我看到一只手高高举起榔头、即将敲下的一刻，我感受到身体内部的张力——那是一种在行动与克制之间的悬停。

（此处可放作品局部 / 过程图）

我想捕捉这种「力量被凝固」的瞬间，于是选择用榔头与钉子来构建这个立体结构。榔头本应是主导的工具，但在这里，它却被定格在钉子的世界中——它失去了攻击性，反而被钉子们支撑着、包围着。

我将所有物件喷涂成统一的奶白色，是想消解它们原本的功能属性，让它们从「工具」变成「形体」。那些亮橙色的细节——钉头与榔头的一面——像是残余的能量痕迹，是力的轨迹、温度的提示。

（此处可放作品展示图）

整体结构中，榔头的倾斜、钉子的密集与规律，制造出一种几乎要倒下却又稳固的平衡感。这像是身体与理性之间的关系：既有冲动，又有克制；既有重量，又有支撑。

暂停的力量

在这件作品里，我呈现的不是一个动作，而是力量在时间中被暂停的状态。

#02 — Memory of the Flesh (2025/11/13)
Concept · 作品阐述

《皮囊记忆 #02 / Memory of the Flesh #02》延续了“皮囊”与“记忆”这一主题，以极简的形式探讨身体、时间与自我意识的关系。镜面被框入洁白的边界中，像一张被档案化的证据页。观者在镜中看到的倒影，与文字——“皮肤替我记得一切，时间在皮囊下沉睡。”——重叠在一起，形成一种被动的对视关系。


画框象征“被社会规训的目光”，而镜面是“自我反思的现场”。当观众靠近镜面时，他们既成为观看者，也成为被观看者。文字以冷静的黑体呈现，如科学记录，却在含义上充满感性：皮肤不再只是生理的界面，而是一层储存疼痛、温度与时间的“活体记忆”。


作品编号“#2”与日期“2025/11/13”，让整件作品带有档案、记录与实验的意味——仿佛是一段关于“身体记忆”的研究样本，也是一份艺术家在特定时间留下的存在证据。
